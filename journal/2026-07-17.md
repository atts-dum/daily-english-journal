## Begin a Tropical Night

### Original 🤔
Last night, before going to bed, I set the air conditioner on a 2-hour timer.  
Until now, I've been able to sleep soundly until morning, but last night I woke up twice, so I had to reset my timer.    
This marks the start of this season's tropical nights.  
I’ve actually lived in Osaka, which used to be called “subtropical.” That’s no big deal!
