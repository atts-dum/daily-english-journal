## Audible While Driving

### Original 🤔
Lately, I’ve been listening to audiobooks while working, that is, while driving.  
As a result, work efficiency, that is, the number of deliveries per hour has decreased.  
It might be dangerous if I don’t concentrate on driving.  
But this is so interesting!  
By the way “У войны не женское лицо”, that’s so thick I gave up on reading it.  

### Corrected 🤖
Lately, I’ve started listening to audiobooks while I’m working—that is, while I’m driving.  
Because of that, the number of deliveries I make per hour has gone down.  
It might be dangerous if I don’t concentrate on driving.  
But this is really interesting!  
By the way, the book is *War Has No Woman’s Face*—a thick tome I’d given up on reading before.
